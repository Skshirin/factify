Factify – Fake News & Hate Speech Detection Using AI

Factify is an AI powered Web Application that analyzes text and detects whether the content is Fake News or contains Hate Speech.
This project uses a DistilBERT Transformer model trained for text classification tasks.

Features:-
Fake News Detection,
Hate Speech Detection,
Real-time analysis,
Modern UI with smooth UX,
Fast Python backend API

Tech Stack:-
Frontend - React + Vite + TailwindCSS + Framer Motion,
Backend -	Python Flask,
AI Model -	DistilBERT Transformer,
Language -	Python + TypeScript

How to Run Locally
Frontend:-
npm install,
npm run dev

Backend:-
cd backend,
pip install -r requirements.txt,
python app.py

Future Scope:-
Image & Video fake detection
